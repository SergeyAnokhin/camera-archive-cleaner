# Руководство пользователя — Camera Archive Cleaner

Camera Archive Cleaner — это удобный инструмент для управления архивами снимков и видеозаписей с камер наблюдения. Ниже описан типовой сценарий использования приложения, горячие клавиши и ключевые настройки.

---

## 1. Типовой сценарий очистки архива

Чтобы очистить архив от гигабайт «мусора» (ложных срабатываний на ветер, дождь, тени, птиц или пауков), следуйте этому стандартному сценарию:

```
[Сканирование] ➔ [Анализ теплокарты] ➔ [Просмотр часа] ➔ [Режим изменений] ➔ [Выделение] ➔ [Удаление]
```

### Шаг 1: Сканирование (Scan)
Для актуализации информации нажмите кнопку **Scan** в правом верхнем углу интерфейса. Приложение просканирует каталоги выбранной камеры на диске и запишет метаданные новых файлов в базу данных.
*Совет: Сканирование работает инкрементально и занимает всего несколько секунд для новых файлов.*

### Шаг 2: Анализ теплокарты (Heatmap)
Теплокарта позволяет визуально находить периоды аномальной активности.
- Ячейки с более насыщенным цветом содержат большее количество файлов/объёма.
- Нажмите на ячейку года, месяца или дня, чтобы спуститься ниже по иерархии (двойной клик или клавиша `Enter`).
- Обращайте внимание на цветные баджи (**AF / SE / BC**) в правом углу ячейки часа — это метрики автоматического обнаружения ложных тревог.

### Шаг 3: Просмотр часа (Hour Viewer)
Кликните на конкретный час, чтобы открыть список файлов. Здесь вы увидите сетку эскизов фотографий и видеозаписей за этот час, разбитых по страницам.

### Шаг 4: Выделение изменений (View Mode)
По умолчанию включен обычный режим просмотра (`Normal`). Чтобы быстро отсеять статический фон и заметить реальные изменения:
- В верхнем выпадающем меню переключите режим на **Motion highlight** (сравнение с часовым средним) или **Erosion** (интеллектуальный фильтр шума).
- На снимках подцветятся только те зоны, где действительно произошло движение. Это позволяет за секунды пролистать сотни кадров.

### Шаг 5: Выделение файлов (Selection)
Если в выбранном часе зафиксирован «мусор» (например, 200 кадров качающейся ветки):
- Нажмите кнопку **Select** (или клавишу `Space`) на первом файле.
- Кликните **Select All** на панели управления внизу, чтобы выделить все файлы на странице, либо выделите диапазон, зажав клавишу `Shift`.

### Шаг 6: Безопасное удаление (Delete)
Нажмите кнопку **Delete** в нижней панели выделения.
- Откроется окно подтверждения удаления.
- Приложение покажет список удаляемых файлов и запустит воспроизведение парного видео (если файлы привязаны к видеозаписи), чтобы вы могли убедиться в безопасности операции.
- Подтвердите удаление. Файлы будут навсегда удалены с физического диска (NAS / SMB-шара), освобождая место.

## 2. Справочник горячих клавиш (Keyboard Shortcuts)

Использование клавиатуры значительно ускоряет разбор архива:

| Клавиша | Действие на теплокарте | Действие в просмотре часа |
| :--- | :--- | :--- |
| `↑` `↓` `←` `→` | Перемещение курсора по ячейкам | Перемещение курсора по файлам |
| `Enter` | Войти в выбранный период (дрель-даун) | Открыть оригинал фото / воспроизвести видео |
| `Esc` / `Backspace` | Подняться на уровень выше (назад) | Закрыть плеер / вернуться к теплокарте |
| `Space` | Включить режим выделения ячеек | Выделить текущий файл |
| `Shift` + клик | — | Выделить диапазон файлов от предыдущего к текущему |
| `Ctrl` + `A` | Выделить все ячейки на текущем уровне | Выделить все файлы на текущей странице |
| `Delete` | Удалить выбранные ячейки (только для часов) | Удалить выделенные файлы |

## 3. Индикаторы ложных тревог (Метрики равномерности)

Приложение рассчитывает три метрики равномерности распределения файлов для выявления циклической записи:

1. **AF (Active Fraction)** — доля минут в часе, содержащих хотя бы одну запись. 100% означает, что камера писала каждую минуту.
2. **SE (Shannon Entropy)** — энтропия Шеннона. Показывает степень хаотичности записей. Высокое значение указывает на равномерную циклическую запись (ветер, дождь).
3. **BC (Block Coverage)** — покрытие 5-минутных блоков.

**Цвета баджей на ячейках часов:**
- **Зелёный**: Единичные события (например, прошёл человек). Записи сосредоточены в узком интервале времени.
- **Жёлтый / Красный**: Высокая равномерность. Скорее всего, это ложная тревога. Рекомендуется открыть этот час и удалить записи.

## 4. Настройки камер и бэкенда

- **База данных**: Все камеры теперь хранятся непосредственно в базе данных. Вы можете добавлять, редактировать и удалять их в меню **Tools ➔ Cameras** (Инструменты ➔ Камеры).
- **Синхронизация**: Ваши настройки интерфейса (размер шрифта, лимиты, burst-интервалы) сохраняются на сервере при закрытии окна **Tools**. При переходе на другое устройство или в другой браузер они подгрузятся автоматически.
- **Ключи API**: В целях безопасности ключи API для Gemini и Claude хранятся исключительно локально в вашем браузере (`localStorage`) и никогда не отправляются на сервер.
